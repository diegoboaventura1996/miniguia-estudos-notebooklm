# miniguia-estudos-notebooklm

Desafio DIO: Caderno temático NotebookLM

CONTEXTO E OBJETIVOS: O tema escolhido para o caderno temático é "Fundamentos de Marketing", o qual é também uma das minhas disciplinas do período atual da graduação em Marketing. O objetivo é criar materiais que auxiliem na revisão dos conteúdos estudados, como resumos e questões.

CURADORIA DE FONTES:

As fontes de conhecimento usadas no caderno, que incluem materiais acadêmicos e guias completos sobre o tema, serão listadas abaixo:

ENGENHARIA DE PROMPTS: Os prompts inseridos por mim dentro do NotebookLM para interagir com a base de conhecimento visam facilitar a organização, visualização e entendimento dos conceitos principais e derivados. A seguir estão os comandos utilizados e seus respectivos arquivos contendo os resultados entregues pela ferramenta:

Prompt para resumo geral do conteúdo: "Faça um resumo estruturado sobre fundamentos do marketing, organizando em parágrafos curtos e tópicos os conceitos principais, conceitos derivados e suas explicações"

Resultado entregue: [prompt_resumo_geral.pdf](https://github.com/user-attachments/files/26413207/prompt_resumo_geral.pdf)


Prompt para comparação detalhada entre dois conceitos: "Faça um tabela comparando os conceitos de Inbound e Outbound marketing por suas vantagens e desvantagens"

Resultado entregue: [prompt_comparativo.pdf](https://github.com/user-attachments/files/26413208/prompt_comparativo.pdf)


Prompt para glossário de termos importantes: "Crie um glossário com os termos que definem as principais métricas de desempenho usadas no marketing"

Resultado entregue: [prompt_glossario.pdf](https://github.com/user-attachments/files/26413209/prompt_glossario.pdf)
